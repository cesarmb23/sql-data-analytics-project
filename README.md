# sql-data-analytics-project

Este repositorio es la continuación del proyecto de **sql-data-warehouse**, en esta parte vamos a hacer todo la **exploración, análisis y reportes de datos** utilizando SQL.

Estos scripts abarcan los siguientes tipos de análisis:

1. Exploración de Base de Datos.
2. Exploración de Dimensiones.
3. Exploración de Fechas.
4. Exploración de Medidas.
5. Exploración de Magnitud.
6. Exploración de Ranking.
7. Tendencias de Cambios en el Tiempo.
8. Análisis Cumulativo.
9. Análisis de Rendimiento.
10. Análisis de Proporción
11. Segmentación de Datos.
12. Reportes. 

Este repositorio contiene consultas SQL diseñadas para mostrar los diferentes tipos de análisis qué podemos realizar cómo analistas de datos y profesionales de Business Intelligence, segmentar y analizar rápidamente los datos dentro de una base de datos relacional.

Cada Script se centra en un tema análitico específico y muestra las mejores prácticas para las consultas SQL.
